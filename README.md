# Exercícios de Lógica com TypeScript

Repositório com exercícios de lógica resolvidos usando **TypeScript**.

## Exercícios

- Par ou Ímpar
- Maior Número
- Soma de Array
- Contagem de Letras
- Nome Reverso
- Tabuada
- Média de Array
- Remover Duplicatas
- Palíndromo
- FizzBuzz

## Tecnologias

- TypeScript
- Node.js

## Objetivo

Praticar lógica de programação e preparação para desenvolvimento profiddional para ultilização de outras ferramentas como o **React** entre outros.
